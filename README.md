# Zhang Shengnan Portfolio

Static deployment package for Zhang Shengnan's portfolio site.

## Deploy on Cloudflare Pages

- Framework preset: None
- Build command: leave empty
- Output directory: `site`

This repository already contains the built static site and optimized portfolio assets.
Large video files are stored outside the Cloudflare output directory and loaded from GitHub media URLs because Cloudflare Pages limits individual deployed files to 25 MiB.
